# Software as a Service (SaaS)

SaaS stellt fertige Anwendungen direkt über den Browser zur Verfügung.  
Der Nutzer muss keine Installation oder Wartung übernehmen.

## Merkmale
- Sofort nutzbar
- Wartung und Updates durch Anbieter
- Zugriff von überall

## Beispiele
- Microsoft 365
- Google Workspace
- Salesforce
